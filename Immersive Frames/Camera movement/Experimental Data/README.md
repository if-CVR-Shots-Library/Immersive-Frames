 Dataset for SAM/VRSQ responses (Camera Movement in CVR)
Content:
	•	This folder provides experimental materials and de-identified questionnaire data for the manuscript “Investigating the Emotional Impact of Camera Movement in Cinematic Virtual Reality.”
	•	Data file: CVR movement_Data_.xlsx
	•	Conditions: asynchronous (ACM), synchronous (SCM), asynchronous with visual fixation (ACM-F)
	•	SAM: 9-point ratings for pleasure (valence), arousal, and dominance, recorded after each stimulus clip
	•	VRSQ: five-level ratings (None at all, Slight, Moderate, Significant, Extreme), recorded after each stimulus clip
	•	Instrument documentation: SAM and VRSQ.docx
	•	The dataset is anonymized (no personal identifiers are included).Dataset for SAM/VRSQ responses
